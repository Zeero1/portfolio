Project Overview

For my Full-Time Semestral Project, I developed a module and dashboard to detect and monitor rogue devices on the network. This project enhances an existing system by building a comprehensive module that identifies unauthorized devices. It utilizes the Django web framework to develop an asynchronous dynamic dashboard, enabling for real-time monitoring and device management. The system is implemented by using a M5StickC device for data collection and Raspberry Pi for edge processing.

Video Demonstration

The video displays the dashboard operating in real-time. The user can see the list of devices and their real-time status through the dashboard.
